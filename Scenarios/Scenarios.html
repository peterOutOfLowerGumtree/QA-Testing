Scenarios

(Site: twitter.com)

  1. Load homepage
  2. Log in

3. User logging into website to upload a profile picture

(Site: twitter.com)

  1. Load homepage
  2. Log in

  4. Undo work

    2. Log out

(Site: twitter.com)

  1. Load homepage
  2. Log in

  4. Follow user
  5. Message user
  6. Undo work

    2. Log out
